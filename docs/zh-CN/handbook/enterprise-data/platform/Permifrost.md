## Permifrost

![permifront 示意图](../../images/handbook/enterprise-data/platform/pf.jpeg)

Permifrost 是一个用于管理 Snowflake 数据仓库权限的 Python 工具。使用该工具的主要文档可在项目和 PyPI 上找到。

### 注意事项

#### ✅ Permifrost 是什么以及它的功能

  * 仅支持 Snowflake
  * Permifrost 仅处理角色和权限
  * 配置文件中未包含的现有对象不会导致错误
  * 是一个完全开源的包，任何人都可以贡献
